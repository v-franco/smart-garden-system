Control de electrovalvula #1

MODO MANUAL:

Ingrese '1' para encender la electrovalvula 1 segundo

Ingrese '2' para apagar la electrovalvula

Ingrese '3' para desconectar el servidor

Ingrese '6' para abrir electrovalvula indefinidamente

MODO AUTO:

Ingrese '4' Para activar modo automático

Ingrese '5' Para desactivar modo automático

Modos auto y manual deben ser operados independientemente, no en conjunto

No se recomienda apagar el servidor a menos que ya no se vaya a utilizar el control del dispositivo

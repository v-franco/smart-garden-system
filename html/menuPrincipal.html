Descripción del reto

El reto consiste en realizar un sistema de riego casero operado por dispositivos conectados entre sí empleando IoT.
Para ello se utilizaron dos tipos de dispositivos: La raspberry Pi y el NodeMCU.
La Raspberry Pi se encuentra como la computadora host de un DDNS, permitiendole desplegar una página web, la cual se comunica con los scripts necesarios (CGIs y Python) para controlar los sistemas de riego, así como recibir datos de sensores conectados al NodeMCU utilizando MQTT.
Los datos obtenidos por los distintos sensores, así como los registros de encendido de las electrovalvulas, son almacenados en una base de datos MySQL.
